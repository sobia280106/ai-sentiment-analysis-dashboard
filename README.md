# AI-Based Sentiment Analysis of Social Media Posts

## Overview
This project uses Natural Language Processing (NLP)
and Machine Learning to analyze sentiments and emotions
from social media posts.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP
- TF-IDF
- LinearSVC (SVM)

## Features
- Bulk social media analysis
- Pie chart visualization
- Emotion analytics dashboard
- Interactive colorful UI
- Machine learning-based classification

## How to Run

Install libraries:

pip install -r requirements.txt

Run app:

streamlit run app.py
